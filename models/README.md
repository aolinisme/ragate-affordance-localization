# Models Directory

This directory is ignored by git except for this README.

Place or symlink local model assets here before running heavyweight
experiments:

```text
models/
  dinov3/
  dinov3_vits16_pretrain_lvd1689m.pth
  FLUX.1-Kontext-dev/
  sd-turbo/
```

Official model sources:

- DINOv3-S/16: https://huggingface.co/facebook/dinov3-vits16-pretrain-lvd1689m
- DINOv3 code: https://github.com/facebookresearch/dinov3
- FLUX.1 Kontext-dev: https://huggingface.co/black-forest-labs/FLUX.1-Kontext-dev
- SD-Turbo: https://huggingface.co/stabilityai/sd-turbo
